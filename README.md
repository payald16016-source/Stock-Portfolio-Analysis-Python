# Live Stock Portfolio Tracker 📈
This project tracks live stock market data and visualizes portfolio health.

## Key Features:
- *Live Data:* Fetched using yfinance API.
- *Visualizations:* Pie charts for allocation and Bar charts for Profit/Loss.
- *Tools:* Python, Pandas, Matplotlib, Seaborn.

## Why this project?
As a career-switcher from Finance to Data Science, I built this to solve real-world investment tracking problems using automation.
